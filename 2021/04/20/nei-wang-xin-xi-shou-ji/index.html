内网信息收集

1.WMIC 基础语法

1.1 语法介绍

wmic命令行是用来管理WMI系统的， 算是win的插件 ， 用户可以使用 WMI 管理本地和远程计算机。

WMI在渗透测试中不需要下载和安装，是Windows系统自带功能，和cmd一样用

在整个运行过程都在计算机内存中发生，不会留下任何操作痕迹。

1.2 简单实用

1.2.1 查看帮助

wmic /?  #查看全局命令
process /? #进程管理帮助
wmic process get /?   #属性获取操作帮助

1.2.2 常用命令

#获取系统正在运行的进程、进程文件的路径、进程ID
wmic process get caption,executablepath,processid

#获取系统当前正在运行的服务
wmic service where (state='runing') get name,processid,pathname,startmode,caption

#查看服务信息
wmic service list brief

#获取杀软详情
wmic /Node:localhost /namespace:\\root\securitycenter2 path antivirusproduct GET displayName,productState,pathToSignedProductExe /Format:List

#获取补丁信息
wmic qfe get Caption,Description,HotFixID,InstalledOn

#判断是否是虚拟机
wmic onboarddevice get Description,DeviceType,Enabled,Status /format:list

#安装了那些软件
wmic product get name,version

#获取本地安装程序
wmic /NAMESPACE:"\\root\CIMV2" PATH Win32_Product get name /FORMAT:table

#系统环境变量
wmic environment get Description,VariableValue

#计算机品牌，型号、名称、名字
wmic computersystem get Name,Domain,Manufacturer,Model,Username,Roles /format:list

#获取系统驱动情况
wmic sysdriver get Caption,Name,PathName,ServiceType,State,Status /format:list
#查看驱动器
wmic logicaldisk get caption,description,providername  

#查看当前操作系统
wmic OS get Caption,CSDVersion,OSArchitecture,Version

#查看开机自启动
wmic startup get caption,command

#查看共享文件
wmic share get name,path,status

1.2.3 操作用户

#搜索用户账户
wmic useraccount list brief
#锁定用户账号
wmic useraccount where name='hacker' set disabled=false
#用户账号重命名
wmic useraccount where name='hacker' rename hacker
#限制用户修改密码
wmic useraccount where name='hacker' set passwordchangeable=false
#搜索登录用户
wmic logon list brief
#搜索计算机域控制器
wmic ntdomain list brief

1.2.4 操作服务

#卸载和重装程序
wmic product where "name like '%forit%'" get name

#停止运行服务,主要是关闭杀软
Wmic process where name="XXX.exe" call terminate
wmic process where name="explorer.exe" call terminate

1.2.5 清理日志

wmic nteventlog where filename='system' cleareventlog

2. 本机信息

2.1 介绍

当主机上线时候，第一件事就是搜集有用信息，判断这是什么系统、找杀软、找漏洞来提权、判断内网等等。

而不是无头苍蝇，上来就乱搞一通。

2.2 命令

2.2.1 系统与配置信息

#主机信息
hostname 

#查看当前操作系统，重点查看版本、位数、补丁信息
systeminfo 
systeminfo | findstr /B /C:"Os Name" /C:"OS Version"

#查看系统架构
echo %PROCESSOR_ARCHITECTURE%

#查看驱动情况
DRIVERQUERY
#查看磁盘驱动器，有几个磁盘
wmic logicaldisk get caption,description,providername

#host 信息
type C:\\windows\system32\drivers\etc\hosts

#如果安装iis服务器可以查看iis配置信息
type C:\\windows\system32\inetsrv\config\applicationHost.config

2.2.2 软件与进程信息

#主要是看安装软件和版本信息，查响应版本漏洞
wmic product get name,version
#查看当前进程列表和进程用户，分析软件、邮件客户端、VPN和杀软等
tasklist /svc
wmic process list brief
#查看AV安装情况
wmic /Node:localhost /Namespace:\\root\SecurityCenter2 Path AntiVirusProduct Get displayName /Format:List
#已启动程序的信息，查看开机自启动
wmic startup get command,caption

2.2.3 服务信息

#查看启动的服务
net start
#查看服务信息
q
#查询服务的详细信息
sc querys

2.2.4 计划任务

schtasks #计划任务
参考参数
/Create  # 创建计划任务
/Delete  #删除计划任务
/Query #显示所有的计划任务
/Change  #更改计划任务
/Run。# 运行计划任务
/End。#终止当前计划任务
/ShowSid  #显示与计划任务名称相应的安全标识符
/？  #显示此帮助消息

#查询所有的计划任务
schtasks /query /fo list /v

2.2.5 用户信息

#查看主机开机时间
net statistics workstation
#查询已经在线用户，注意管理员是否在线
query user || qwinsta
query session 
#查看当前用户权限，主要是判断现在是本地平台用户、本地管理员、还是域内用户
whoami
whoami /all
# 查看本地用户列表，推测命名规则
net user
#查看所有组名，
net localgroup
#然后可以查看指定组中的成员列表，比如查询本地管理员组看到本地一个管理员，还一个域的Domainadmins为域内机器的本地管理员用户
net localgroup administrators

2.2.6 网络配置信息

#查看网络信息
ipconfig /all
#查看路由器和路由表，可以判断内网的地址段
route print
arp -A
#查看主机共享列表，可访问的共享列表
net share
wmic share get name,path,status
#查看主机开放端口以及链接状态，TCP/UDP
netstat -ano
netstat -anob #查看对应的程序，需要权限

#通过注册表查看代理情况
REG QUERY "HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings"

3域信息收集

3.1 域内信息收集

3.1.1主机信息收集

shell systeminfo #查看主机信息，存在域信息
#获取域名ip
shell nslookup god.org
ping god.org 

#查询当前会话登陆的是域还是本地工作组，是域的话可以查DNS和域用户信息
shell config workstation

3.1.2域基本信息查询

shell net view #查看工作组内主机
shell net view/domain #查看有几个域
shell net view/domain：god.org  #查看指定域内主机
shell net accounts /domain   #获取域密码策略
shell nltest /domain_trusts #获取域信任信息

3.1.3.域内用户和管理员信息

net user /domain #查询域内用户列表
net user Administrator /domamin # 查询指定域用户信息
net group /doamin  #查看域内的组，判断每个组的功能
net group "domain admins" /domain #查询域管理员组内的用户
net group "Domain computers" /domain  #查询域成员计算机列表，对应的是主机的机器账户

3.1.4判断主域（域控）

net time /domain #域服务器一般会作为时间服务器使用，就可以获得域控地址存在域，但当前用户不是域用户会拒绝访问；是域用户就获得相应地址；显示找不到域则是工作组环境
ping DC.domain.test #通过ping获取域控
nltest /DCLIST:domain
netdom query pdc
net group "Domain controllers" /domain # 查看域控制器的机器账户
